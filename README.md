# PsychLogic Mock Test Portal — Assets

This folder is reserved for future assets (fonts, icons, etc.)

## Files in Root:
- index.html — Main portal (all tests, login, dashboard, exam, results)
- testdata.js — Question data for Mock Tests 1, 2, 3
- logo.png — PsychLogic brand logo
- CNAME — Your custom domain (edit this file with your actual domain)

## GitHub Pages Setup:
1. Create a new repository on GitHub
2. Upload all files (including this _assets folder)
3. Go to Settings → Pages → Branch: main → Root (/)
4. Your domain: edit CNAME with your actual domain

## Adding Tests 4 & 5:
When ready, the testdata.js will be updated with TEST4_DATA and TEST5_DATA.
